## YomiLens (요미렌즈) : 일본어 학습 보조 도구 앱 **YomiLens(요미렌즈)**
가로쓰기 일본어 한자의 후리가나(읽는 법)를 실시간으로 확인하여, 한자 부수를 그리지 않고도 빠르게 단어를 검색할 수 있도록 돕는 앱입니다.

## TapTap Golf (탭탭골프) : 애플워치용 골프스코어 관리 앱 **TapTap Golf(탭탭골프)**
골프 라운딩 중 스코어를 기록하는 앱으로, 애플워치의 간단한 조작으로 기록을 남길 수 있는 것이 핵심 기능입니다.
